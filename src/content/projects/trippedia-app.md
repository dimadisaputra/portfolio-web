## Project Overview

Trippedia App is a desktop-based application developed for client offer management purposes at the Trippedia Perkasa company. This application is designed to simplify and accelerate the process of creating travel proposals, whether for custom offers, open trips, or private trips. The goal is to improve operational efficiency by providing a centralized tool to manage all aspects related to travel offers.

## Key Responsibilities

- **Core Feature Development**: Designing and building the core functionality of the application to create, edit, and dynamically manage various types of travel packages.
- **Data Management**: Implementing modules to display essential information such as the company profile, transportation fleet details, and a complete history of clients and their orders for quick access.
- **User Interface (UI) Design**: Building an interactive and responsive user interface using React, with efficient state management using Redux.
- **Desktop Architecture**: Using Electron.js to build a cross-platform desktop application from a web-based codebase (JavaScript, HTML, CSS).
- **Local Database**: Designing and managing a local database using SQLite to securely store all offer data, client information, and other essential details on the user's device.
